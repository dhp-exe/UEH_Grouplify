# UEH Grouplify Frontend

Mobile-first student group-finding frontend built with React, TypeScript, Vite, TailwindCSS, and React Router.

## Tech Stack

- React (functional components + hooks)
- TypeScript
- Vite
- TailwindCSS
- React Router
- Mock data only (no backend/API)

## Run Locally

1. Install dependencies:

   npm install

2. Start development server:

   npm run dev

3. Build for production:

   npm run build

## Routing Flow

- /: Personal profile
- /find-group/menu: Find group options
- /find-group/loading: Matching loading screen
- /find-group/analysis: Class skill analysis
- /find-group/suggestions: Suggested groups
- /chat/flareup: Group chat screen
- /explore: Explore students screen
- /tasks: Personal tasks screen
